Creamos la clase Arreglo 2D aleatorio con margenes. Esta clase nos creara una configuracin inicial aleatoria con margenes de n* densidad y de N particulas. Esta clase tiene varios mtodos o funciones que definiran al objeto que estamos programando.

  • Init: El mtodo init es la funcion que se correra automaticamente cuando iniciames la clase. Consta de pedirnos los parametros n* y N.
  • L: Nos calcula la longitud de la celda con la formula: $$ L = \sqrt{ \dfrac{N}{n^*} } $$
  • r: Nos calcula el radio de las particulas con la formula: $$ r = \sqrt{ \dfrac{n^*}{2}} $$
  • new_point: Nos genera un nuevo punto que este dentro de los margenes de la celda. Para eso se tiene que generar un numero tomando en cuanto el largo de la longitud y multiplicarla por $1- \dfrac{2r}{l}$ ya que esta generara un punto que no sobrepasara los extremos de la caja.
  • array: Este es el metodo que generara las N particulas aleatorias en la caja sin traslapes. Primero genero un punto nuevo y lo guardo en una lista. Se crea un ciclo que vaya hasta las N particulas, se genera un nuevo punto con la funcin new_point y se guarda en la lista creada anteriormente. Despues se crea otro ciclo que vaya hasta donde termine el contador de la funcin anterior, esto servira para comparar la distancia de la particula nueva con las generadas anteriormente. Si la distancia es mayor al diametro $2r$ de las particulas entonces que no haga nada pero si es menor, que elimine el ultimo punto de la lista y le reste al contador del ciclo. Por ultimo que me devuelva la lista generada.
  • graph: Esta funcin nos permitira grficar las N particulas. En esta funcin solo se guardaran las graficas.
  • graph_show: Esta funcin tambien nos permite grficar las N particulas. Pero aqu si se mostraran en la terminal.

class arreglo2D_aleatorio_cm:
    def __init__(self,n,N):
        self.n=n
        self.N=N
        self.data=self.array()
        self.large = self.L()

    def L(self):
        return (self.N/self.n)**(1/2)

    def r(self):
        return float(np.sqrt(self.n)/2)
    
    def new_point(self):
        x=self.L()*(1-2*self.r()/self.L())*(random.random() - 1/2)
        y=self.L()*(1-2*self.r()/self.L())*(random.random() - 1/2)
        return (x,y)

    def array(self):
        
        Dat=[self.new_point()]
        
        i=0
        while len(Dat)<self.N:
            i = i+1
            x=self.L()*(1-2*self.r()/self.L())*(random.random() - 1/2)
            y=self.L()*(1-2*self.r()/self.L())*(random.random() - 1/2)
            Par=(x,y)
            Dat.append(Par)
            for j in range(1,i+1):
                distancia=distance.euclidean(Par,Dat[i-j])
                if distancia < 2*self.r():
                    Dat.pop()
                    i=i-1
                    break
        return Dat

    def graph(self,data,title,size,c,contador,trazadora):
        color_pre=c

        fig, ax = plt.subplots(figsize=(size,size))

        plt.grid(linestyle='--')
        ax.set_aspect(1)
        for i in range(0,len(data)):
            if i == trazadora:
                c="r"
            else:    
                c=color_pre
            circle1 = plt.Circle(data[i], self.r(), color=c)
            ax.add_artist(circle1)

        plt.title(title, fontsize=16)
        plt.savefig("./image/" + str(contador) + ".png")
        plt.close(fig)

class montecarlo2D(arreglo2D_aleatorio_cm):

    def mod_pot(self, distancia):
        if distancia < self.large/2 and distancia < 2*self.r():
            V =  1e10
        else:
            V = 0
        return V

    def sumaup(self,data):
        V = 0
        i=1
        m=0
        while i<len(data):
            for j in range(0,i):
                ri=np.array(data[i])
                rj=np.array(data[j])
                ri = ri - self.large*np.round(ri/self.large)
                rj = rj - self.large*np.round(rj/self.large)
                rij= ri - rj  
                distancia=np.linalg.norm(rij)
                Vij = self.mod_pot(distancia)
                V = V + Vij
            i = i+1           
        return V

    def UP(self,i,ri,data):
        V = 0

        for j in range(0,len(data)):
            if i != j:
                ri = ri - self.large*np.round(ri/self.large)
                rj=np.array(data[j])
                rj = rj - self.large*(np.round(rj/self.large))
                rij= ri - rj  
                distancia=np.linalg.norm(rij)
                Vij = self.mod_pot(distancia)                
                V = V + Vij
        return V
